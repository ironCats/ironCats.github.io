一、简明搭建教程

1. 更新软件包索引

2. 安装git，vim，node.js，npm，hexo

3. 初始化hexo

4. 修改配置文件_config.yml

5. 生成、查看、复制ssh密钥

6. 配置密钥至自己的github仓库

  • 在github上创建名为yourname.github.io的仓库，其中yourname是你的github用户名
  • 点击 Setting >> Deploy keys >> Add deploy key，上面的Title框任意填写，下面的Key框粘贴刚才复制的ssh密钥，勾选Allow write access，点击Add Key

7. 安装hexo-deployer-git远程部署插件

8. 生成静态文件，部署到github上
